\doxysubsection{Detailed Description}
\begin{DoxyVerb}The goal is to send first ball 1 and then ball 0 in the holes. If the arm touches ball 1, is considered as failure.
  State is composed of:
  s = ([ball0_x, ball0_y], [ball1_x, ball1_y], [joint0_angle, joint1_angle], [joint0_speed, joint1_speed])

  The values that these components can take are:
  ball_x, ball_y -> [-1.5, 1.5]
  joint0_angle -> [-pi/2, pi/2]
  joint1_angle -> [-pi, pi]
  joint0_speed, joint1_speed -> [-50, 50]\end{DoxyVerb}
